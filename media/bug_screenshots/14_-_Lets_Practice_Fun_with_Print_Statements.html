In this exercise, you'll practice using print statements in Java to create fun and interactive outputs. Use System.out.print and System.out.println to achieve different effects. Include the \\n escape sequence for new lines and basic arithmetic operations directly in the print statements.

Exercise Instructions:

Write Java code for the following tasks:

  1. Print your name on one line.

  2. Print your age on the same line.

  3. Print the result of 5 + 3 on the next line.

  4. Print a statement that says "I am learning Java!" on a new line, with Java in the next line. Use a single print statement for this.

Tips:

  • Use System.out.print for printing without a newline and System.out.println for printing with a newline.

  • Use \\n to create new lines in your output.

  • Incorporate basic arithmetic directly within the print statements.
